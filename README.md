# 🚗 Parkiei - Flutter

Aplicativo mobile desenvolvido em Flutter com foco em simulação de busca e navegação para estacionamentos.

---

## 📌 Descrição

O **Parkiei** é um protótipo de aplicativo mobile que simula a busca por estacionamentos próximos, visualização de rotas e exibição de informações dos locais.

O projeto foi desenvolvido com fins acadêmicos, com foco na prática de construção de interfaces mobile e organização de código utilizando Flutter.

---

## 🎯 Objetivo

O objetivo do projeto é:

* Praticar desenvolvimento mobile com Flutter
* Aplicar conceitos de organização de projeto
* Construir interfaces modernas e funcionais
* Simular um aplicativo real de estacionamento

---

## 📱 Funcionalidades

* Tela de login
* Tela de cadastro
* Listagem de estacionamentos
* Visualização de rota
* Perfil do estacionamento
* Tela administrativa (simulada)
* Modal de iniciar percurso
* Modal de finalizar percurso

---

## 🧠 Conceitos aplicados

* Uso de `StatelessWidget` e `StatefulWidget`
* Organização em pastas (`screens`, `widgets`, `models`)
* Componentização de interface
* Uso de `CustomPainter` para simular mapa
* Layout com `Column`, `Row`, `Stack` e `ListView`
