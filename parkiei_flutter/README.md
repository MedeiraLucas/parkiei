# 🚗 Parkiei Flutter
Aplicativo mobile desenvolvido em Flutter com foco em localização e gerenciamento de estacionamentos.

---

## 📱 Sobre o projeto

O **Parkiei** é um protótipo de aplicativo que simula a busca, visualização e navegação até estacionamentos próximos.

Este projeto foi desenvolvido com o objetivo de praticar:

* construção de interfaces mobile
* organização de código em Flutter



---

## 🎯 Funcionalidades

✔️ Tela de login
✔️ Tela de cadastro
✔️ Listagem de estacionamentos próximos
✔️ Visualização de rota até o destino
✔️ Perfil de estacionamento com avaliações
✔️ Área administrativa (mock)
✔️ Modal de iniciar percurso
✔️ Modal de finalizar percurso

---

## 🧠 Conceitos utilizados

* `StatelessWidget` e `StatefulWidget`
* Organização por pastas (`screens`, `widgets`, `models`)
* Componentização de UI
* CustomPainter (simulação de mapa e rota)
* Layout responsivo básico
* Design system simples (cores, botões e inputs padronizados)
